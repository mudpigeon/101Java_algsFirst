## 技术

### 链表操作

假设 x 是一条链表的某个结点且不是尾结点。则

```java
x.next = x.next.next;
```

删除了 x 的后继结点



假设 x 是一条链表的某个结点, t 是另外一个结点, 则

```java
t.next = x.next;
x.next = t;
```

将 t 插入到 x 的后继。
